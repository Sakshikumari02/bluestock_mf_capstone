# bluestock_mf_capstone
# Mutual Fund Analytics Capstone Project

# Project Overview

This project analyzes mutual fund performance, investor behavior, SIP trends, and risk metrics using Python, SQLite, SQL, and Power BI. The solution includes data ingestion, cleaning, analytics, dashboarding, and fund recommendation capabilities.

# Setup Instructions
Install Dependencies
- pip install -r requirements.txt
- Required Tools
- Python 3.x
- Jupyter Notebook
- SQLite
- Power BI Desktop

# How to Run the ETL Pipeline

Execute the master pipeline script:

python run_pipeline.py

The ETL pipeline performs:
- Data Ingestion
- Data Cleaning
- Data Validation
- Database Loading
- Analytics Processing
- Report Generation

# How to Open the Dashboard
1. Open Power BI Desktop.
2. Navigate to the dashboard folder.
3. Open:
   bluestock_mf_dashboard.pbix
4. Refresh data if required.
5. Explore dashboard pages and filters.

Dashboard Pages:

- Industry Overview
- Fund Performance
- Investor Analytics
- SIP & Market Trends
